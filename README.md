# Django Job Portal API

A backend REST API for a Job Portal built using Django and Django REST Framework.

## Features
- Job creation and listing
- Resume management
- Clean architecture
- Environment variable configuration
- Git version control with proper commit standards

## Tech Stack
- Python
- Django
- Django REST Framework
- SQLite

## Setup Instructions
1. Clone the repository
2. Create virtual environment
3. Install requirements
4. Configure .env file
5. Run migrations
6. Start server
